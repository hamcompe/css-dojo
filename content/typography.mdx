## Punctuation

| name    | symbol | command              |
| :------ | :----- | :------------------- |
| Hyphen  | -      | `-`                  |
| En Dash | –      | option + `-`         |
| Em Dash | —      | option + shift + `-` |

#### When to use ?

- **Hyphen**: It’s primarily used for hyphenating words, hence the name, and for connecting two or more words together: five-dollar bills.
- **En Dash**: Its main purpose is to replace the word “to”, like 5–6 p.m.
- **Em Dash**: The em dash can be used as an indicator of speaker in a narrative. —“Why is typography important?”. It can also be used as an indicator of a break.

### Quotation Marks

Make sure you use the correct curly quotation marks. Straight marks are for coding only.

<p style="font-size: 2rem; font-weight: 500; margin: 0; font-family: 'times new roman'">"Don't do this."</p>
<p style="font-size: 2rem; font-weight: 500; margin: 0; font-family: 'times new roman'">“You’re cool!”</p>
