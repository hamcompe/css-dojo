| Selector          | Example description                                                           |
| :---------------- | :---------------------------------------------------------------------------- |
| element           | Selects elements by element tag                                               |
| #id               | Selects by id                                                                 |
| .class            | Selects by class                                                              |
| .class1.class2    | Selects elements that contains 2 of these classes                             |
| .class1,class2    | Selects class1 and class2                                                     |
| .class1 .class2   | Selects class2 only inside class1                                             |
| div > p           | Selects p where **parent** is a div                                           |
| div + p           | Selects all `<p>` elements that are placed immediately after `<div>` elements |
| [attribute=value] | Selects all elements with attribute = value                                   |
| p:nth-child(2)    | Selects every `<p>` element that is the second child of its parent            |

**_Resource_**: [w3schools CSS selectors](https://www.w3schools.com/cssref/css_selectors.asp)
