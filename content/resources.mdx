# All the recommended resouces

## Learn

- [flexbox cheatsheet](https://yoksel.github.io/flex-cheatsheet/) — Flexbox interactive cheatsheet.
- [Learn CSS grid with scrimba](https://scrimba.com/g/gR8PTE) — Learn CSS Grid video record.
- [Learn layout](https://learnlayout.com/) — Learn basic CSS display, position for managing your layout.
- [Every layout](https://every-layout.dev/) — Learn how to build most often layout use cases.

## Good read

- [better web type](https://betterwebtype.com/) — Tons of resources about typography.
- [CSS Tricks](https://css-tricks.com/) — As the name said, it’s a blog for CSS.
- [30seconds of CSS](https://30-seconds.github.io/30-seconds-of-css/) — Learn by example of the things you can make by CSS in 30 seconds.
- [Web Content Accessibility Guidelines](https://www.w3.org/TR/WCAG21/) — Accessibility guideline documentary.
